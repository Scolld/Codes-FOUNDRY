# Quest Manager

Gestionnaire de quêtes pour Foundry VTT avec visualisation en arbre et graphe réseau.

## Installation

1. Copier le dossier `quest-manager` dans `foundrydata/Data/modules/`
2. Redémarrer Foundry VTT
3. Activer le module dans votre monde

## Fonctionnalités

- Gestion hiérarchique des quêtes
- Statuts : Connue, En cours, Terminée
- Relations entre quêtes (parent/enfant, bloque/bloqué, lié)
- Visualisation en arbre dépliable
- Visualisation en graphe réseau
- Synchronisation temps réel GM/Joueurs
- Système de permissions granulaires
- Liens vers acteurs, items, journaux dans les descriptions
